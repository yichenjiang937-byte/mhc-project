## Data Files

The raw input files (CSV files and FASTA file) as well as the **Gold result files** generated by the filtering workflow are hosted on [Figshare](https://doi.org/10.6084/m9.figshare.31566175). 

- The **raw files** include:
  - `mhc_ligand_full_*.csv`
  - `human_proteome.fasta`

- The **Gold result files** generated after the filtering process are used for downstream analysis and visualization in the visualizer tool.
