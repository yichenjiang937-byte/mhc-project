# MHC Data Filtering and Visualization Project

This repository contains two connected tools for a two-step MHC data processing workflow:

1. **Filter tool**  
   Processes raw MHC ligand data together with the human proteome FASTA reference and generates filtered Gold result files.

2. **Visualizer tool**  
   Reads the Gold result files generated by the filter step and provides local visualization.
