Visualizer tool

Files:
- app_visualization.py
- main_visualizer.py
- start_visualizer.bat
- build_exe.bat

Usage:
1. Put these two CSV files in the same folder:
   - Class_I_HomoSapiens_Gold_Gene.csv
   - Class_II_HomoSapiens_Gold_Gene.csv
2. Double-click start_visualizer.bat to run.
3. Or run build_exe.bat on Windows to build MHC_Visualizer.exe.
